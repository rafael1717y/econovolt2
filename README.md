Um simulador de consumo de energia elétrica.

### 1. Para instalar

#### LINUX

```console
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

#### WINDOWS

```console
python3 -m venv .venv
source .venv/Scripts/activate
pip install -r requirements.txt
```

### 2. Para executar

```console
export FLASK_ENV='development'
export FLASK_SECRET_KEY='uma-senha-secreta'
flask run
```
